## Prerequisites

Install Rust:

```bash
rustup install stable
```

Verify installation:

```bash
rustc --version
cargo --version
```

## Running Programs

Compile and run any Rust source file:

```bash
rustc filename.rs
./filename
```

Example:

```bash
rustc arrays.rs
./arrays
```

## Using the Makefile

Compile:

```bash
make
```

Clean generated executables:

```bash
make clean
```
